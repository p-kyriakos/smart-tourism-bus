# Smart Tourism Bus

A React + TypeScript web application that simulates a **Smart Tourism Bus** experience. The application provides dedicated interfaces for passengers, drivers, and staff, featuring order management, GPS navigation, tourist attraction information, and basic bus system simulations.

## Features

- Select a user role:
  - Passenger
  - Driver
  - Staff

### Passenger Dashboard
- Driver's front-view simulation
- Tourist attractions display
- Coffee ordering system
- GPS navigation
- Demo payment flow

### Driver Dashboard
- Speed monitoring
- Driver fatigue level
- Door control
- System alerts and notifications

### Staff Dashboard
- Cabin temperature control
- Roof control
- Solar panel monitoring
- Robotic vacuum management

## Technologies Used

- React
- TypeScript
- Vite
- React Router
- CSS

## Getting Started

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

## Production Build

Create a production build with:

```bash
npm run build
```

## Project Purpose

This project was developed as a demonstration of a smart tourism bus management system, showcasing role-based interfaces and interactive features for passengers, drivers, and staff in a modern web application.
